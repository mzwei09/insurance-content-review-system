# 前端页面截图指南

本文档提供详细的截图步骤，帮助生成 README 所需的界面预览图。

## 📸 快速截图方法

### 方式 A：使用系统截图工具（推荐）

**macOS**：
- 截取选定区域：`Cmd + Shift + 4`
- 截取整个窗口：`Cmd + Shift + 4`，然后按`空格`，点击窗口
- 截图会自动保存到桌面

**Windows**：
- 使用 Snipping Tool 或 `Win + Shift + S`
- 截图后保存到指定位置

**Linux**：
- 使用 `gnome-screenshot` 或 `flameshot`

### 方式 B：使用浏览器开发者工具

1. 按 `F12` 打开开发者工具
2. 按 `Cmd/Ctrl + Shift + P` 打开命令面板
3. 输入 "screenshot"
4. 选择 "Capture full size screenshot"

## 🎨 截图技巧

1. **窗口大小**：调整浏览器窗口为 1920x1080，确保内容完整显示
2. **内容清晰**：确保文字清晰可读，避免模糊
3. **隐藏敏感信息**：
   - API 密钥应该显示为掩码（`sk-****...`）
   - 可以使用测试邮箱（如 `test@example.com`）
4. **统一风格**：所有截图使用相同的浏览器和窗口大小
5. **完整性**：确保重要元素都在截图范围内
