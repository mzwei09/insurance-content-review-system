# 前端页面截图指引

本文档说明如何为 README 生成所需的前端页面截图。

## 截图规格

- **格式**：PNG
- **分辨率**：1920×1080 或更高
- **保存路径**：`docs/screenshots/`

## 截图清单与操作步骤

### 1. 登录页面 (`01-login.png`)

**操作步骤：**
1. 启动服务：`bash start.sh`
2. 打开浏览器访问 http://localhost:8000
3. 若已登录，先点击右上角「退出登录」
4. 确保当前显示为登录表单（标题「登录账号」）
5. 将浏览器窗口调整为 1920×1080 或更大
6. 截取整个页面，保存为 `docs/screenshots/01-login.png`

**页面特征**：AI 图标、标题「保险营销内容智能审核」、用户名/密码输入框、「登录」按钮、「还没有账号？ 立即注册」链接

---

### 2. 注册页面 (`02-register.png`)

**操作步骤：**
1. 在登录页面点击「立即注册」
2. 确保显示注册表单（标题「注册新账号」）
3. 截取整个页面，保存为 `docs/screenshots/02-register.png`

**页面特征**：用户名、密码、邮箱（可选）输入框、「注册」按钮、「已有账号？ 立即登录」链接

---

### 3. 审核页面 - 空状态 (`03-review-empty.png`)

**操作步骤：**
1. 使用已注册账号登录（若无账号，先完成注册）
2. 登录成功后自动进入审核页面
3. 确保左侧输入区为空，右侧显示「输入文本或上传图片后点击"开始审核"」
4. 截取整个页面，保存为 `docs/screenshots/03-review-empty.png`

**页面特征**：左侧输入框和图片上传区为空，右侧审核结果区显示空状态提示

---

### 4. 审核页面 - 文本审核 (`04-review-text.png`)

**操作步骤：**
1. 在审核页面左侧输入框中输入测试文本，例如点击「夸大收益」示例按钮填充
2. 点击「开始审核」按钮
3. 等待审核完成（约 5–15 秒）
4. 确保右侧显示完整审核结果（合规/违规、违规类型、置信度、引用条文、审核理由）
5. 截取整个页面，保存为 `docs/screenshots/04-review-text.png`

**页面特征**：左侧有输入文本，右侧显示审核结果卡片（含合规状态、违规类型、置信度条、引用条文列表、审核理由）

---

### 5. 审核页面 - 图片审核 (`05-review-images.png`)

**操作步骤：**
1. 在审核页面点击或拖拽上传多张图片（可使用 `test_images/` 目录下的测试图片）
2. 点击「开始审核」按钮
3. 等待审核完成（多图约 12 秒/张）
4. 确保右侧显示「详细审核结果」，包含每张图片的审核结果
5. 截取整个页面，保存为 `docs/screenshots/05-review-images.png`

**页面特征**：左侧显示已上传图片预览，右侧显示多张图片的详细审核结果（每张图片的合规/违规、违规类型、理由）

---

### 6. 个人中心页面 (`06-profile.png`)

**操作步骤：**
1. 在已登录状态下，点击顶部导航栏「个人中心」
2. 确保显示 API 密钥配置界面
3. 截取整个页面，保存为 `docs/screenshots/06-profile.png`

**页面特征**：账号信息（用户名、邮箱）、百炼 API 密钥输入框、保存/验证按钮、当前状态显示

---

## 快速截图流程（推荐顺序）

1. 启动服务 → 退出登录 → 截 01、02
2. 登录 → 截 03（空状态）
3. 输入文本并审核 → 截 04
4. 上传图片并审核 → 截 05
5. 进入个人中心 → 截 06

## 自动截图脚本（可选）

项目提供了 Playwright 自动截图脚本，需先安装依赖：

```bash
pip install playwright
playwright install chromium
```

在服务已启动的前提下运行：

```bash
python scripts/capture_screenshots.py
```

脚本会依次截取 6 张截图并保存至 `docs/screenshots/`。若未配置 API 密钥，文本/图片审核相关截图可能跳过。

## 注意事项

- 截图前请确保 API 密钥已配置，否则文本/图片审核可能失败
- 若使用开发者模式（`.env` 配置密钥），可跳过个人中心密钥配置步骤
- 建议使用无痕/隐私模式或清除 localStorage，以确保登录页和注册页为干净状态
