## 🧪 使用方法

### 方法1：Web界面测试

1. 启动系统：`bash start.sh`
2. 登录并配置API密钥
3. 在审核页面上传测试图片
4. 点击"开始审核"
5. 查看审核结果是否符合预期

### 方法2：API接口测试

```bash
# 获取JWT token（先注册登录）
TOKEN="your_jwt_token_here"

# 测试图文审核
curl -X POST http://localhost:8000/api/review-multimodal \
  -H "Authorization: Bearer $TOKEN" \
  -F "text=请审核这张保险营销海报" \
  -F "images=@1_违规_夸大收益.png"
```

## 💡 测试技巧

### 1. 测试纯图片审核

- 只上传图片，不输入文本
- 系统会自动提取图片文字并审核

### 2. 测试图文混合审核

- 上传图片 + 输入文本
- 系统会将图片文字和输入文本合并审核

### 3. 测试多图审核

- 同时上传2-3张图片
- 系统会提取所有图片的文字并合并审核

## 📝 注意事项

1. **图片质量**：图片文字清晰度会影响OCR效果
2. **API密钥**：需要配置有效的百炼API密钥
3. **网络连接**：需要能访问阿里云Dashscope API
4. **处理时间**：图文审核比纯文本审核稍慢（2-5秒）
