# 测试用例说明文档

## 概述

本目录包含保险营销内容审核系统的评估测试用例，用于验证审核模型对各类合规/违规内容的识别能力。

## 文件说明

- `test_cases.json`: 测试用例主文件，每项包含：
  - `id`: 用例唯一标识
  - `content`: 待审核的营销内容文本
  - `expected_compliance`: 预期是否合规 (true/false)
  - `expected_violation_type`: 预期违规类型（合规时为 null）
  - `expected_articles`: 预期引用的监管条文编号列表
  - `description`: 用例说明

## 合规样本

合规样本共 20+ 个，覆盖：
- 标准产品介绍（重疾险、意外险、年金险、医疗险、寿险）
- 带风险提示的表述
- 边界合规案例（机构评级、客户反馈、持证推荐等）
- 备案与条款阅读提示

## 复杂场景

- **多种违规混合**：case_042（夸大收益+不当比较）、case_043（无资质代言+虚假宣传+夸大收益）、case_044（误导+不当比较）
- **边界模糊表述**：case_045、case_046、case_048

## 使用方法

```bash
# 运行评估脚本
python scripts/run_evaluation.py
```

评估报告将生成在 `reports/` 目录下。
